# LIGA A 

Liga A. - это небольшой адаптивный лендинг с использованием grunt, html (+ БЭМ нейминг), scss и js 

Для запуска:

1. Клонируем репозиторий
 ```
 git clone https://github.com/cool-hiya/liga-a.git
 ```
2. Устанавливаем node_modules
 ```
 npm install
 ```
3. Запускаем
 ```
 npm run start
 ```
